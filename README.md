# hrishikeshpatil_bitsom_ba_2511159_part1_data_cleaning
Part 1 – Business Data Cleaning, Validation & Excel Reporting
Problem Summary
This project involved cleaning and validating a retail orders dataset exported from multiple business systems. The main goal was to identify data quality issues, apply appropriate business rules, prepare a clean and reliable dataset for analysis, and create useful Excel reports for the business team.
Dataset Description
The dataset contains retail order transactions, including customer details, product information, sales amounts, discounts, shipping information, payment status, and order status.
Data Cleaning Steps
Kept the original raw dataset untouched and worked on a separate cleaned version
Standardized text formatting (categories, regions, ship modes, etc.)
Corrected inconsistent category and sub-category names
Validated shipping dates (flagged cases where Ship Date occurred before Order Date)
Removed exact duplicate records
Flagged invalid or problematic records
Data Quality Summary
Original Records: 932
Final Clean Records: 912
Exact Duplicate Records Removed: 20
Data Retention Rate: 97.85%
Key Business Insights
Duplicate records were identified and removed, significantly improving data reliability
